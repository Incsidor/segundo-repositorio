Exemplos de usos de tags Por Paulo

descrito por Paulo ASP AlphaED Tech

html e a linguagem basica para a criacao de uma pagina web, usamos algumas tags como


head: para construir um cabecario

title para dar um nome a sua pagina

body:para o corpo da pagina

footer: para os pes da pagina ou melhor dizendo o final dela

img: para inserir uma imagem

h1,h2,h3, h4 e etc,para adicionarmos hierarquia os textos

p: e ultilizado para textos simples

br: usado para separar as linhas sem o uso de um paragrafo

ul:usado para criar uma lista

li: usado para criar conteudo da lista

a: serve para criar um texto com link
